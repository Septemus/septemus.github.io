> # 使用教程

打开`Visual Studio`:

![step1](/images/step1.png)

选择克隆存储库

![step2](/images/step2.png)

粘贴`github`源代码仓库地址：[https://github.com/Septemus/swjtu-computergraphics-exp](https://github.com/Septemus/swjtu-computergraphics-exp.git)

![step3](/images/step3.png)

设置好路径后点击克隆

![step4](/images/step4.png)

点击启动按钮

![step5](/images/step5.png)

启动成功

![step6](/images/step6.png)
